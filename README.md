# Choose your chew

I will create an application in which you can calculate your optimal number of calories per day. Here you can create a personalized menu, which will indicate the calorie content of each product and the total amount of calories for breakfast, lunch and dinner. You can also find information about the composition of products and their properties.
The goal of this project is to make mindful eating enjoyable and easy.
As an extension of this project in the future, I would like to create an opportunity to share dishes and save the ones you like on your page.
